Send SMS from your existing Contact Form 7 plugin using SMS Global.

== Description ==

Send SMS from your existing Contact Form 7 plugin using SMS Global.

You need to have SMS Global account(http://www.smsglobal.com) and some credit to use this plugin.

== Languages Available ==

== Installation ==

1. Install and activate the plugin on the Plugins page
2. Need to have SMS Global Account for sending SMS

== Frequently Asked Questions ==

= What to do if you need to change the plugin's code? =
When you changed the plugin's code you should also change the plugin's version to '100' (for example) to avoid updates, which could override and delete your code.
